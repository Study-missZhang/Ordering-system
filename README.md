# Ordering-system
这是一个使用Java基础编写的点菜系统。
程序需求：
			1、首先主函数运行显示一个主界面：
							*****主菜单*****
							菜单			1
							已点菜单		2
							结账			3
			2、点击1进入菜单页面：
							****请您点菜****
							1		鱼香肉丝 38.0
							2		糖醋里脊 28.0
							3		宫保鸡丁 31.5
							4		清蒸鲈鱼 42.0
							5		烩三鲜	 23.5
			3、输入1/2/3...显示已点菜单：
							您已点：****
			4、按0返回上一个页面
			5、点击2显示已点菜单
							*****您已点*****
							烩三鲜   23.5
							宫保鸡丁 31.5
			6、点击3结账
							****正在结账，请稍后****
							共消费了：55.0元
